# 库？框架？

## 库和框架的差别

> Don't call Us, We'll call You.

### 库

- 小巧
- 专注于解决某个特定的问题或提供特定的功能：DOM操作、动画效果等
- 通常被应用程序代码调用：它听我们的
- 灵活

### 框架

- 大而全
- 成套的解决方案：数据绑定、组件化、路由、状态管理
- 通常去调用我们程序代码：我们得听它的
- 有它自己的规则和规范

## VueJS与ReactJS

它们核心都是库。

不过在库之上又提供了不同方面的工具与方法。

## 总结

- 框架是包含了库以及相关附加功能的解决方案
- 现主流框架的原理
- 都是基于状态渲染视图
